## AI-Powered ETL Data Quality Monitor

## Overview
Intelligent ETL pipeline monitor that auto-detects 
data quality issues and generates fix suggestions using LLaMA 3.

## Tech Stack
- **LLM:** LLaMA 3 via Groq API
- **Data Processing:** Python, Pandas
- **Use Case:** ETL Pipeline Anomaly Detection

## Features
- Auto-detects nulls, duplicates, outliers
- LLM-generated root cause analysis
- Natural language remediation suggestions
